About the Game

This is a minimalist interpretation of the classic Snake game. Each character represents a different personality, expressed through visual style and behavior.

How to Play

Select a character, enter their page, and control the snake using your keyboard or touch gestures. The goal is simple: survive.

Why This Game Exists

Built as a calm, distraction-free experience inspired by old arcade logic and editorial design. No noise. No clutter.
